# Plans

Implementation plans for standard-tooling work items. Plans are
organized by lifecycle stage:

| Directory | Contents |
|---|---|
| `proposed/` | Plans drafted but not yet under active implementation. |
| `in-progress/` | Plans with active implementation work underway. |
| `completed/` | Plans whose implementation is finished. |

Move a plan from `proposed/` to `in-progress/` when implementation
begins, and from `in-progress/` to `completed/` when all tasks are
done. Specs remain in `docs/specs/` and are not affected by plan
lifecycle.
