* feat(docker): decouple standard-tooling from dev container images (#362)

Add per-branch Docker image caching so non-Python repos get
standard-tooling installed once at image build time rather than
pre-baked into dev container images. st-docker-run auto-builds cached
images on first use; hash-based invalidation rebuilds when
st-config.toml or lockfiles change.
